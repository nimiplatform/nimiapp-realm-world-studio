# Failure Semantics

- **[R-RWS-FAIL-001]** Realm unavailable.
- **[R-RWS-FAIL-002]** Permission missing.
- **[R-RWS-FAIL-003]** Public world detail unavailable.
- **[R-RWS-FAIL-004]** Public world-character cards unavailable.
- **[R-RWS-FAIL-005]** Public media unavailable.
- **[R-RWS-FAIL-006]** Showcase projection payload invalid.
- **[R-RWS-FAIL-007]** Relation or chat capability unavailable.
- **[R-RWS-FAIL-008]** Runtime transport unavailable.
- **[R-RWS-FAIL-009]** Runtime route unbound.
- **[R-RWS-FAIL-010]** Runtime output malformed.
- **[R-RWS-FAIL-011]** Public world sections incomplete.
- **[R-RWS-FAIL-012]** Session UI state unavailable.
- **[R-RWS-FAIL-013]** No failure may be converted into placeholder success, fake return data, synthesized metric, or renderer-local authority.
