# Failure Semantics

- **[R-RWS-FAIL-001]** Realm unavailable.
- **[R-RWS-FAIL-002]** Runtime transport unavailable.
- **[R-RWS-FAIL-003]** Permission missing.
- **[R-RWS-FAIL-004]** Creator world inventory unavailable.
- **[R-RWS-FAIL-005]** WorldCore detail unavailable.
- **[R-RWS-FAIL-006]** WorldCharacterCore list/detail unavailable.
- **[R-RWS-FAIL-007]** Realm write rejected or content hash stale.
- **[R-RWS-FAIL-008]** Typed DTO payload invalid.
- **[R-RWS-FAIL-009]** Kit UI/model-config capability unavailable.
- **[R-RWS-FAIL-010]** Runtime AI route unbound or output malformed.
- **[R-RWS-FAIL-011]** Asset/resource resolver unavailable.
- **[R-RWS-FAIL-012]** Session UI state unavailable.
- **[R-RWS-FAIL-013]** No failure may be converted into placeholder success, fake return data, synthesized metric, or app-local authority.
