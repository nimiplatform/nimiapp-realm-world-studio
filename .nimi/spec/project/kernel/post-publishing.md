# Post Publishing Boundary

- **[R-RWS-POST-001]** RWS does not own public post publishing for world characters in the current kernel.
- **[R-RWS-POST-002]** Runtime-generated or creator-authored social post copy is draft/candidate material until a future typed Realm publishing surface is admitted.
- **[R-RWS-POST-003]** RWS must not create app-local schedule, campaign, moderation, or publication authority.
- **[R-RWS-POST-004]** Any future post surface must define author source, Realm write owner, attachment envelope, review state, scheduling semantics, and fail-closed behavior before implementation.
