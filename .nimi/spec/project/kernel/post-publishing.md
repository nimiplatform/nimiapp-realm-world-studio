# Post Publishing Boundary

- **[R-RWS-POST-001]** World Atlas does not own public post publishing for world characters in the current kernel.
- **[R-RWS-POST-002]** Runtime-generated or creator-authored post copy is not part of this page.
- **[R-RWS-POST-003]** World Atlas must not create local schedule, campaign, queue, or moderation authority.
- **[R-RWS-POST-004]** Any future post surface must define author source, Realm write owner, attachment envelope, review state, and failure semantics before implementation.
